Problem

https://onlinejudge.u-aizu.ac.jp/courses/lesson/2/ITP1/2/ITP1_2_A

Small, Large, or Equal

Write a program which prints small/large/equal relation of given two integers a and b.

Input

Two integers a and b separated by a single space are given in a line.

Output

For given two integers a and b, print

a < b
if a is less than b,

a > b
if a is greater than b, and

a == b
if a equals to b.

Constraints

-1000 ≤ a, b ≤ 1000

Sample Input

Sample Input 1
1 2
Sample Input 2
4 3
Sample Input 3
5 5

Sample Output

Sample Output 1
a < b
Sample Output 2
a > b
Sample Output 3
a == b

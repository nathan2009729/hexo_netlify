更換開機logo

1
sudo apt-get install gimp

2 change picture size to 800*480

open gimp 

圖片 -> 縮放圖片-> 800*480

圖片 -> 模式->索引色-> 224

3 save as PPM format#使用匯出功能，非另存新檔

-> PPM format
    -> ASCII format

4 copy to kernel

cp ~/logo.ppm ~/linux-3.5-Tiny4412/drivers/video/logo

5 remove/rename the original logo ppm file and replace the logo with out logo

把新的檔案取代原本開機會使用的ogo_linux_clut224.ppm 

cd ~/linux-3.5-Tiny4412/drivers/video/logo

mv logo_linux_clut224.ppm logo_linux_clut224_old.ppm

mv logo.ppm logo_linux_clut224.ppm

6 compile kernel and upload the zImage to Tiny4412

6-1 cd ~/linux-3.5-Tiny4412

6-2 make -j4

6-3 cd arch/arm/boot

6-4 板子端在bootloader階段輸入fastboot

6-5 sudo fastboot flash kernel zImage

6-6 板子端按重開機鍵，可在螢幕顯示新圖片

安裝小蜜蜂(aliens)

小蜜蜂是sdl project的遊戲，可利用以下指令下載到

wget http://www.libsdl.org/projects/aliens/src/aliens-1.0.2.tar.gz

安裝SDL_mixer-1.2.11

網路上下載SDL_mixer-1.2.11的tar.gz檔，用terminal解壓縮後進入目錄。以下是編譯步驟：

/configure —host=arm-linux CC=arm-linux-gcc —prefix=/usr/local/arm/4.5.1/arm-none-linux-gnueabi —with-sdl-prefix=/usr/local/arm/4.5.1/arm-none-linux-gnueabi

make -j4

sudo -s

make install

exit

cp /usr/local/arm/4.5.1/arm-none-linux-gnueabi/lib/mixer /home/nathan/root_mkfs/lib

cp -rf /usr/local/arm/4.5.1/arm-none-linux-gnueabi/sys-root/lib/libSDL_mixer-1.2.so* /home/nathan/root_mkfs/lib

安裝libmikmod-3.1.11

網路上下載libmikmod-3.1.11的tar.gz檔，用terminal解壓縮後進入目錄。以下是編譯步驟：

export CC=arm-linux-gcc

./configure —host=arm-linux —prefix=/usr/local/arm/4.5.1/arm-none-linux-gnueabi

make

sudo -s

make install

編譯小蜜蜂(alien)

把下載回來的aliens-1.0.2.tar.gz給解壓縮，進到目錄厚用終端機執行以下指令：

export CC=arm-linux-gcc

./configure —host=arm-linux —with-sdl-prefix=/usr/local/arm/4.5.1/arm-none-linux-gnueabi

make -j4

cp aliens /home/nathan/root_mkfs

cp -r data /home/nathan/root_mkfs

到板子端，執行./aliens即可再板子螢幕上看到遊戲畫面。

完

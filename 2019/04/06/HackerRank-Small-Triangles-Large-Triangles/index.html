要注意的是，我們的目的只是比大小，不需要算出精確的面積。因此，我們只要比較兩人面積的平方，這樣不但可以正確地比較，也可以避免開根號的麻煩。
海龍公式我們可以寫成$S^2 = p \times (p-a)\times(p-b)\times(p-c)$，所以，只要比較 $p\times(p-a)\times(p-b)\times(p-c)$的值(在int compare_area(const void *a, const void *b)函式裡)，再利用C標準庫中的qsort來排序即可。
